# .scout — AI Agent Workspace

AI-native engineering context for coding agents working on this codebase.

## Structure

```
.scout/
├── agents/    — Agent definitions (roles, responsibilities, guards)
├── skills/    — Reusable skill documents for specific tasks
└── prompts/   — Shared prompt templates (reserved)
```

## Usage

- Agents define specialized AI personas for different engineering roles
- Skills provide structured workflows for common tasks
- Prompts hold reusable prompt templates for LLM interactions

See `AGENTS.md` in the project root for the complete engineering context.
